# Process modern Census Day format (2024+) CDE data

Process modern Census Day format (2024+) CDE data

## Usage

``` r
process_enr_modern(raw_data, end_year)
```

## Arguments

- raw_data:

  Raw data frame from get_raw_enr_modern()

- end_year:

  The school year end for context

## Value

Processed data frame with standard schema
