# Build CDE cumulative enrollment file URL

Constructs the download URL for cumulative enrollment file (alternative
data source). Cumulative enrollment counts all students enrolled during
the year, not just Census Day.

## Usage

``` r
build_cumulative_enr_url(end_year)
```

## Arguments

- end_year:

  A school year end (e.g., 2024 for 2023-24 school year)

## Value

URL string
